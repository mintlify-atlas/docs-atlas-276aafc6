## Version 1.1.3 (Current)

<Note>
This is the latest stable release of Open Screen.
</Note>

### Features

- **Project Save/Load** - Save and load project files to continue editing later
- **Speed Adjustment** - Control playback speed with configurable keyboard shortcuts
- **Keyboard Shortcuts Configuration** - Customize keyboard shortcuts with conflict detection
- **Cursor Telemetry-Driven Zoom Suggestions** - Intelligent zoom suggestions based on cursor movement
- **Enhanced Export Options** - Improved encoder queue management and export performance

### Improvements

- Zoom focus clamping for better control
- Annotation bounding and canvas wrapping
- Timeline UX quality-of-life improvements
- Keyframe snap and move functionality
- Better latency mode for improved throughput

### Bug Fixes

- Fixed configurable shortcuts for speed adjustment
- Improved encoder queue management
- Fixed zoom suggestion logic organization

## Version 1.1.2

### Features

- **Google Fonts Integration** - Access to Google Fonts for text annotations
- **16:10 Aspect Ratio Support** - Added support for 16:10 aspect ratio exports
- **System Tray Icon** - Dynamic tray icon and menu updates

### Improvements

- Anti-aliasing enabled for smoother rendering
- Timeline UX improvements with better keyframe handling
- Content clamp component to show clamped text
- Optimized clamped content display

### Bug Fixes

- Fixed border radius appearing smaller in export compared to preview
- Fixed Windows close button not working in HUD overlay
- Fixed motion blur now disabled by default for better performance
- Fixed source selector tabs content height

### Documentation

- Added Full Disk Access instructions for macOS
- Added troubleshooting for Linux sandbox error
- Moved issue templates to YAML forms
- README cleanup and improvements

## Version 1.1.0

### Features

- **GIF Export** - Export your recordings as GIF files with customizable settings
- **Tutorial Help Component** - In-app guidance for video trimming
- **Linux Support** - Full Linux support with AppImage distribution

### Improvements

- Custom hidden scrollbar styles
- Adjusted layout and sizing of settings panel tabs
- Reduced Linux AppImage boot time from ~50s to near-instant
- Improved gitignore configuration

### Bug Fixes

- Fixed GIF export options (removed problematic FPS settings)
- Fixed minimum height for image selection tab in settings panel
- Fixed background image selection height issues

### Developer Experience

- Added Biome formatter and linter (replaced ESLint)
- Improved build configuration
- Added support for Windows, macOS, and Linux builds

## Version 1.0.2

### Features

- **Hex-Based Color Inputs** - Improved color picker with hex input support
- **User Selection Control** - Enhanced UI interaction controls

### Improvements

- Demuxer and CFR (Constant Frame Rate) conversion
- Timeline UX improvements
- Better content organization

### Bug Fixes

- Build fixes and optimizations
- Path resolution improvements

## Version 1.0.1

### Initial Public Release

First stable release of Open Screen with core features:

- Screen and application recording
- Manual zoom controls with customizable depth
- Video cropping capabilities
- Background customization (wallpapers, colors, gradients, images)
- Motion blur effects
- Text, arrow, and image annotations
- Video trimming
- Multiple aspect ratios and resolutions
- Export functionality

## Earlier Versions

### Beta Releases (v0.1.0 - v0.2.0)

Early development versions with core recording and editing functionality.
